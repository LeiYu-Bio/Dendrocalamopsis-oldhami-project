group-ID*_vs_group-ID*----------------------------对应分组的基本分析目录
├── enrichment------------------------------------差异富集分析目录
├── heatmap---------------------------------------对应分组差异代谢物热图
├── opls------------------------------------------对应分组正交偏最小二乘判别分析
├── pls-------------------------------------------对应分组偏最小二乘判别分析
├── pca-------------------------------------------对应分组主成分分析
├── cpdCorr---------------------------------------对应分组差异代谢物相关性图
├── TopFcMetabolites------------------------------对应分组差异倍数前N个物质含量差异图
├── vipscore--------------------------------------对应分组差异代谢物VIP值图
├── vol-------------------------------------------对应分组火山图
├── zScore----------------------------------------对应分组差异代谢物Z值
├── group-ID*_vs_group-ID*_info.xlsx--------------对应分组所有代谢物相关信息
├── group-ID*_vs_group-ID*_filter.xlsx------------对应分组差异代谢物相关信息
└── group-ID*_vs_group-ID*_VIP.xlsx---------------对应分组所有代谢物正交偏最小二乘判别分析计算结果


对应分组所有代谢物相关信息/对应分组差异代谢物相关信息表格说明：
Index：迈维ID
Formula：物质分子式
Compounds：物质英文名称
物质：物质中文名称
Class I：物质英文一级类别
物质一级分类：物质中文一级类别
Class II：物质英文二级类别
物质二级分类：物质中文二级类别
CAS：物质CAS号
Level：物质鉴定级别（1：样本物质二级质谱、RT与数据库物质匹配得分为0.7分以上；2：样本物质二级质谱、RT与数据库物质匹配得分为0.5-0.7分；3：样本物质Q1、Q3、RT、DP、CE与数据库物质核对一致）
VIP：变量重要性投影
p_value：显著性检验p值
FDR：多重假设检验验证后的错误发现率
Fold_Change：差异倍数
log2FC：差异倍数以2为底取对数
Type：代谢物上下调类型

注：根据筛选条件不同，表格的列有差异，样本生物学重复不满足分析时，某些列名会不在表格中。
